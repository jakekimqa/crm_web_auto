## Phase 1: 필수값 누락 시 저장 버튼 비활성화 검증

필수값: 이름, 사진, 시술업종(헤어/네일), 기본가격, 담당자(1명 이상)

| # | 테스트 케이스 | 입력 상태 | 기대 결과 |
|---|---|---|---|
| 1-1 | 초기 상태 (전체 누락) | 아무것도 입력 안 함 | 저장 버튼 disabled |
| 1-2 | 이름 누락 | 사진+업종+가격+담당자 입력, 이름만 비움 | 저장 버튼 disabled |
| 1-3 | 사진 누락 | 이름+업종+가격 입력, 사진 미업로드 | 저장 버튼 disabled |
| 1-4 | 시술업종 누락 | 이름+사진+가격 입력, 업종 미선택 | 저장 버튼 disabled |
| 1-5 | 기본가격 누락 | 이름+사진+업종 입력, 가격 비움 | 저장 버튼 disabled |
| 1-6 | 담당자 누락 | 전체 입력 후 "전체 선택 해제" 클릭 | 저장 버튼 disabled |

- 각 케이스 간 뒤로가기 시 "변경 사항이 저장되지 않았습니다." alert → 확인 → 폼 초기화
