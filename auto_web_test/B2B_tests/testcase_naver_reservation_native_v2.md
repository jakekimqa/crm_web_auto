## Phase 0: 로그인 + 샵 선택

### TC-0.1: 로그인 후 네이버테스트 60분 샵 진입

**Step 1: 앱 실행 + 로그인**
- 동작: 앱 실행 (clearState) → 로그인 화면 진입
- 동작: 아이디 입력 필드 탭 → "herrenail" 입력
- 동작: 비밀번호 입력 필드 탭 → "gong2023@@" 입력
- 동작: "로그인" 버튼 탭
- 대기: 샵 선택 화면 표시 (timeout: 10s)

**Step 2: 샵 선택**
- 동작: 샵 목록에서 스크롤하며 "네이버테스트 60분" 탐색
- 동작: "네이버테스트 60분" 텍스트가 보이면 해당 영역의 "샵으로 이동" 탭
- 대기: 캘린더 화면 표시 (timeout: 10s)

**검증:**
- [MUST] 화면에 "네이버테스트 60분" 텍스트 표시
- [MUST] 캘린더(예약) 화면 진입 확인 — "예약" 탭 활성 상태

**Maestro 힌트:**
```yaml
- scrollUntilVisible:
    element: "네이버테스트 60분"
    direction: DOWN
- tapOn: "샵으로 이동"
```

## 검증 레벨 정의

| 레벨 | 의미 | 실패 시 |
|------|------|---------|
| **[MUST]** | 핵심 검증 — 실패 시 테스트 FAIL | 즉시 중단 |
| **[SHOULD]** | 권장 검증 — 실패 시 WARNING | 로그 남기고 계속 |
| **[CAPTURE]** | 스크린샷만 — 검증 없음 | 항상 PASS |
