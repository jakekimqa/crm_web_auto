## 사전 조건

- 테스트 시작 전 해당 날짜의 고객/매출/예약 데이터가 **초기화(클렌징)** 되어 있어야 함

## 배치 대기 처리 가이드

통계 배치가 5분 간격으로 실행되므로, 매출 등록 직후 통계가 즉시 반영되지 않을 수 있습니다.

**권장 구현:**
1. 직원 통계 페이지 진입 → 오늘 필터 적용
2. 샵주테스트 행의 **매출 건수**가 **7건**인지 확인
3. 7건 미만이면 **30초 대기** → 페이지 새로고침 → 재확인
4. 최대 **12회 반복** (총 6분) 후에도 미달이면 **FAIL** 처리

```
MAX_RETRIES = 12
INTERVAL = 30초
EXPECTED_SALES_COUNT = 7

for attempt in 1..MAX_RETRIES:
    직원 통계 페이지 진입 + 오늘 필터
    매출 건수 = 샵주테스트 행의 매출 건수 값
    if 매출 건수 >= EXPECTED_SALES_COUNT:
        break  // 배치 반영 완료
    30초 대기 후 재시도
```
